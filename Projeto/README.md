##  Como Executar

### Pré-requisitos
Para executar este projeto, é necessário ter o **Python 3** instalado e a biblioteca **PLY** (Python Lex-Yacc).

### Utilização
```
python main.py <pasta_input> \[pasta_output]
```

## Testes exemplo
Na pasta input tem vários exemplos da programas em pascal, e na pasta output tem os respetivos resultados em código para a MV
